# Installation

Learn how to install and set up Lumionix.

## Prerequisites

Before installing, ensure you have:

- Node.js 14 or higher
- npm or yarn
- Git (optional)

## Installation Steps

1. Install via npm:

   ```bash
   npm install lumionix
   ```

2. Configure your settings:
   ```bash
   npx lumionix-config
   ```
